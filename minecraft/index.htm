Resource Packs Size Notes
MC-ResourcePack-Blindness.zip ~220KB Replaces every texture with solid #000 1x1 pixel texture.
MC-ResourcePack-Pixel.zip ~8KB Reduces some textures to solid color 1x1 pixel textures.
MC-ResourcePack-Template.zip ~1KB Template for creating a new resource pack.
MC-ResourcePack-TransparentGravel.zip ~2KB Gravel texture is transparent.
To install these resource packs, download and copy the directory of the resource pack you wish to install into your Minecraft "resourcepacks" directory. Deleting a directory from your Minecraft "resourcepacks" directory will uninstall that resource pack.

To begin building your own resource pack from scratch:
  1. Make a copy of the "MC-ResourcePack-Template" directory.
  2. Modify how your pack is displayed in the Minecraft resource packs UI:
    1. Rename the copied directory to the name of your resource pack.
    2. Modify the description in the "pack.mcdata" file.
    3. Modify the "pack.png" image file.
  3. Add the resources you wish to include in the resource pack. Make sure you place resources within the proper directory structure.
  4. Place your resource pack directory into the Minecraft "resourcepacks" directory.
  5. Select your resource pack via the in-game resource pack UI.
