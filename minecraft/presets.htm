iterami/Guides.htm/Minecraft/Superflat Presets
Name Preset Notes
Empty
  • Entirely empty world, so cheats are probably required.
  • Use "/setblock 0 100 0 minecraft:bedrock" to create a stable starting point.
  • Use "/tp 0 101 0" to return to your bedrock starting point.
Lava Trap
  • Lava lake hidden under one layer of sand.
  • Structure generation is recommended so you have something to stand on.
  • All the sand falling at once will cause a lot of lag.
Ocean
  • Ocean 111 units deep.
