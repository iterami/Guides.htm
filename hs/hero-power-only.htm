Guides.htm/Hearthstone/Hero Power Only
Hero Power Only is a hypothetical game mode in which each hero is only allowed to use The Coin and their hero power. Hero powers must be used whenever possible, including turn 1 for players that have The Coin. The 30 cards each hero gets are hypothetically impossible to play, which eventually results in fatigue as normal.
