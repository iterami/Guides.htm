iterami/Guides.htm/World of Warcraft/Settings
  • Chat Settings:
    • Combat Log:
      • Increase background opacity to ~75%
      • Rename to "L"
      • Set "Font Size" to "12 pt"
    • General:
      • Increase background opacity to ~75%
      • Rename to "G"
      • Set "Font Size" to "12 pt"
      • Unlock the window and increase size
  • Interface Settings:
    • ActionBars:
      • Check "Bottom Right Bar"
      • Check "Right Bar"
      • Check "Right Bar 2"
    • Buffs and Debuffs:
      • Check "Show All Enemy Debuffs"
      • Uncheck "Consolidate Buffs"
    • Controls:
      • Check "Auto Loot".
    • Raid Profiles:
      • Change "Display Health Text" to "Health Remaining"
      • Check "Display Class Colors"
      • Check "Display Pets"
      • Check "Display Power Bars"
      • Check "Use Raid-Style Party Frames"
      • Uncheck "Display Border"
