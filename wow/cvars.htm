  • CVar names are not case sensitive.
CVar Default Value Recommended Value Notes
alwayscompareitems 0 1
  • If set to 1, hovering over an item in your backpack that can be equipped will display your currently equipped item in that slot.
  • If set to 0, you must hold shift to display item comparisons.
nobuffdebufffilterontarget 0 1
  • If set to 1, displays all buffs and debuffs on your target, which can help you make better decisions during combat.
  • If set to 0, only your own buffs and debuffs will be displayed.
xpbartext 0 1
  • If set to 1, always displays text values on your XP bar, including when displaying reputation or honor.
  • If set to 0, values are only displayed when your character info window is open.
